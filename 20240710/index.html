2-SAT

倒带倒带倒带，别倒带了，面前太多阻碍。

定义

K-SAT 问题指的是对一个包含若干 \(K\) 元约束的系统的求解。当 \(K\ge 3\) 时问题为 NP 完全问题。

2-SAT 问题：对于以下形式的若干个二元约束条件，求解序列 \(A\) 的合法解是否存在：

  1. \(A_i= V\)；
  2. \(A_i = V, \operatorname{if} A_j=V'\)。

其中 \(V,V'\) 为布尔值。

求解

模仿差分约束的处理方式，将问题放到图上。令点 \(i\) 代表 \(A_i\) 取 true，点 \(i'\) 代表 \(A_i\) 取 false，有向边 \(x\to y\) 表示若答案取 \(x\)，则答案必取 \(y\)。

为了方便叙述，不妨假设 \(V=1,V'=0\)。则约束 2 可转化为 \(j'\to i,j\to i'\) 的 两条 连边，表示：若 \(A_j=0\)，则 \(A_i\) 必为 \(1\)；同时，若 \(A_j\ne 0\)（即 \(A_j=1\)），则 \(A_i\ne 1\)（即 \(A_i=0\)）。第二条边很有必要，否则算法会判定 \(j\to i\) 是一个合法的选择。

考虑只存在约束 2 的系统。系统无解当且仅当对于任意 \(i\)，同时存在 \(i\to i'\) 与 \(i'\to i\) 的有向路径，因为此时选取 \(i\) 和 \(i'\) 作为 \(A_i\) 的状态均不存在合法解。

除此之外的情况，若存在 \(i\to i'\) 的有向路径则选取 \(i'\) 作为答案；反之，若存在 \(i'\to i\) 的有向路径则选取 \(i\) 作为答案。若二者皆不存在，说明 \(i\) 的取值对答案合法性无影响，任选其一即可。

考虑向系统中加入约束 1。对于收到 1 约束的某一 \(i\)，即 \(i(i')\) 必须选择（具体取决于 \(V\) 的值），相应地，\(i'(i)\) 必须不选。结合上文寻找答案的方式，对于约束 1，将必须不选者向必须选择者连边即可。

确定方案

求解过程中，对全图进行强连通分量缩点，若存在 \(i\) 与 \(i'\) 属于同一个强连通分量，则系统无解；

否则，在图上按逆拓扑序确定答案——即，选择 \(i\) 与 \(i'\) 中拓扑序较大者。注意到 Tarjan 求解强连通分量的顺序本身就是逆拓扑序，按点所属的强联通分量编号选取较小者即可。

复杂度

即为 Tarjan 复杂度。为 \(O(n + m)\)。因 \(m\) 阶数一般高于 \(n\)，简写为 \(O(m)\)。

B. 2-SAT 问题

https://www.becoder.com.cn/contest/5350/problem/2

进行如下操作转化：

  • 1 AND：二者均只能取 \(1\)；
  • 0 AND：若一者为 \(1\)，则另一者必为 \(0\)；
  • 1 OR：若一者为 \(0\)，则另一者必为 \(1\)；
  • 0 OR：二者均只能取 \(0\)；
  • 1 XOR：一者为 \(0\)，则另一者必为 \(1\)；一者为 \(1\)，则另一者必为 \(0\)；
  • 0 XOR：一者为 \(0\)，则另一者必为 \(0\)；一者为 \(1\)，则另一者必为 \(1\)。

按照如上转化后的约束连边求解即可。

#include <bits/stdc++.h>
const int maxn = 2e6 + 5;
int now, cnt;
std::vector<int> t;
std::vector<int> g[maxn];
int dfn[maxn], low[maxn], col[maxn];
void DFS(int x) {
	dfn[x] = low[x] = ++now;
	t.push_back(x);
	for (auto i : g[x])
		if (!dfn[i]) {
			DFS(i);
			low[x] = std::min(low[x], low[i]);
		}
		else if (!col[i])
			low[x] = std::min(low[x], dfn[i]);

	if (low[x] == dfn[x]) {
		++cnt;
		for (int p; ; ) {
			p = t.back(), t.pop_back();
			col[p] = cnt;
			if (p == x)
				break;
		}
	}
	return;
}
int main() {
#ifdef ONLINE_JUDGE
	std::ios::sync_with_stdio(false);
	std::cin.tie(nullptr);
	std::cout.tie(nullptr);
#endif
	int n, m;
	std::cin >> n >> m;
	for (int a, b, c; m--; ) {
		std::string op;
		std::cin >> a >> b >> c >> op;
		if (op == "AND") {
			if (c) {
				g[a + n].push_back(a);
				g[b + n].push_back(b);
			}
			else {
				g[a].push_back(b + n);
				g[b].push_back(a + n);
			}
		}
		else if (op == "OR") {
			if (c) {
				g[a + n].push_back(b);
				g[b + n].push_back(a);
			}
			else {
				g[a].push_back(a + n);
				g[b].push_back(b + n);
			}
		}
		else {
			if (c) {
				g[a].push_back(b + n);
				g[b].push_back(a + n);
				g[a + n].push_back(b);
				g[b + n].push_back(a);
			}
			else {
				g[a].push_back(b);
				g[b].push_back(a);
				g[a + n].push_back(b + n);
				g[b + n].push_back(a + n);
			}
		}
	}
	for (int i = 1; i <= 2 * n; ++i)
		if (!dfn[i])
			DFS(i);

	for (int i = 1; i <= n; ++i)
		if (col[i] == col[i + n]) {
			std::cout << "IMPOSSIBLE\n";
			return 0;
		}

	std::cout << "POSSIBLE\n";
	for (int i = 1; i <= n; ++i)
		std::cout << (col[i] < col[i + n]) << ' ';
	std::cout << '\n';
	return 0;
}

F. 婚礼

https://www.becoder.com.cn/contest/5350/problem/6

看似需要将必须对坐的一对夫妇作为对应的一对点，其实不然。

注意到需要处理一对偷情者的位置：既可以分坐两侧，又可以同坐新娘同侧。

考虑将某一人坐桌子左 / 右侧的状态作为一对对应点，使一对夫妇状态必定对立即可。

不妨钦定新娘坐左边，对于一对偷情者，当且仅当其中一人坐新娘对面（桌子右侧），另一人有坐新娘同侧（桌子左侧）的要求。

当某人坐桌子左 / 右侧的状态同时属于一个强连通分量时，无解。

#include <bits/stdc++.h>
const int maxn = 125;
int now, cnt;
std::vector<int> t;
std::vector<int> g[maxn];
int dfn[maxn], low[maxn], col[maxn];
void DFS(int x) {
	dfn[x] = low[x] = ++now;
	t.push_back(x);
	for (auto i : g[x])
		if (!dfn[i]) {
			DFS(i);
			low[x] = std::min(low[x], low[i]);
		}
		else if (!col[i])
			low[x] = std::min(low[x], dfn[i]);

	if (low[x] == dfn[x]) {
		++cnt;
		for (int p; ; ) {
			p = t.back(), t.pop_back();
			col[p] = cnt;
			if (p == x)
				break;
		}
	}
	return;
}
int main() {
#ifdef ONLINE_JUDGE
	std::ios::sync_with_stdio(false);
	std::cin.tie(nullptr);
	std::cout.tie(nullptr);
#endif
	for (int n, m; std::cin >> n >> m, n || m; ) {
		for (int i = 1; i <= n; ++i) {
			g[i].push_back(i + 3 * n);
			g[i + 3 * n].push_back(i);
			g[i + n].push_back(i + 2 * n);
			g[i + 2 * n].push_back(i + n);
		}
		g[1 + 2 * n].push_back(1);
		for (int x, y; m--; ) {
			char a, b;
			std::cin >> x >> a >> y >> b;
			++x, ++y;
			if (a == 'h')
				x += n;
			if (b == 'h')
				y += n;
			g[x + 2 * n].push_back(y);
			g[y + 2 * n].push_back(x);
		}
		for (int i = 1; i <= 4 * n; ++i)
			if (!dfn[i])
				DFS(i);

		for (int i = 1; i <= 2 * n; ++i)
			if (col[i] == col[i + 2 * n]) {
				std::cout << "bad luck\n";
				goto outo;
			}

		for (int i = 2; i <= n; ++i)
			std::cout << (i - 1) << (col[i] < col[i + 2 * n] ? 'w' : 'h') << ' ';
		std::cout << '\n';
	outo:
		std::fill(dfn + 1, dfn + 4 * n + 1, 0);
		std::fill(low + 1, low + 4 * n + 1, 0);
		std::fill(col + 1, col + 4 * n + 1, 0);
		for (int i = 1; i <= 4 * n; ++i)
			g[i].clear(), g[i].shrink_to_fit();
		cnt = now = 0;
	}
	return 0;
}

H. 游戏

https://www.becoder.com.cn/contest/5350/problem/8

看到三种选择，想到和上一题相同的将某地拆成 \(A,B,C\) 分别选、不选共 6 个点的做法。

但经过实践发现常数太大在土豆 OJ 上卡不过。这里提供另一种方法：注意到有 \(d\le 8\) 的限制，考虑有何深意。

这里有一个很妙的处理，注意到除 x 以外的地点都只有两种可选车型，考虑用其作为一点的两个对立状态。

而对于 x，可以任意将其视作 a / b / c，表示该地的可用车为 \(BC\)、\(AC\)、\(AB\)。其中任意选取两种即可覆盖 x 地选择车 \(A,B,C\) 的情况。因其数量极少，可以直接枚举将其视作 a 或 b。

对于每一种可能的将所有 x 用 a / b 替换后的最终序列，跑一遍 2-SAT 即可。复杂度 \((2^d\times m)\)。

#include <bits/stdc++.h>
const int maxn = 3e5 + 5;
int s[maxn];
std::vector<int> t, p;
int now, cnt, n, d, m;
std::vector<int> g[maxn];
int dfn[maxn], low[maxn], col[maxn];
std::vector<std::tuple<int, char, int, char> > q;
void DFS(int x) {
	dfn[x] = low[x] = ++now;
	t.push_back(x);
	for (auto i : g[x])
		if (!dfn[i]) {
			DFS(i);
			low[x] = std::min(low[x], low[i]);
		}
		else if (!col[i])
			low[x] = std::min(low[x], dfn[i]);

	if (low[x] == dfn[x]) {
		++cnt;
		for (int p; ; ) {
			p = t.back(), t.pop_back();
			col[p] = cnt;
			if (p == x)
				break;
		}
	}
	return;
}
bool DFS1(int x) {
	if (x == d + 1) {
		now = cnt = 0;
		std::fill(dfn + 1, dfn + 2 * n + 1, 0);
		std::fill(low + 1, low + 2 * n + 1, 0);
		std::fill(col + 1, col + 2 * n + 1, 0);
		for (int i = 1; i <= 2 * n; ++i)
			g[i].clear();
		int x, y, a, b;
		for (auto i : q) {
			std::tie(x, a, y, b) = i;
			if (s[x - 1] == a)
				continue;
			if (s[y - 1] == b) {
				if ((a - s[x - 1] + 3) % 3 == 1)
					g[x].push_back(x + n);
				else
					g[x + n].push_back(x);
			}
			else {
				if ((a - s[x - 1] + 3) % 3 == 1) {
					if ((b - s[y - 1] + 3) % 3 == 1)
						g[x].push_back(y), g[y + n].push_back(x + n);
					else
						g[x].push_back(y + n), g[y].push_back(x + n);
				}
				else {
					if ((b - s[y - 1] + 3) % 3 == 1)
						g[x + n].push_back(y), g[y + n].push_back(x);
					else
						g[x + n].push_back(y + n), g[y].push_back(x);
				}
			}
		}

		for (int i = 1; i <= 2 * n; ++i)
			if (!dfn[i])
				DFS(i);

		for (int i = 1; i <= n; ++i)
			if (col[i] == col[i + n])
				return 0;

		for (int i = 1; i <= n; ++i)
			std::cout << (char)('A' + (s[i - 1] + (col[i + n] < col[i])) % 3);
		std::cout << '\n';
		return 1;
	}
	s[p[x - 1] - 1] = 1;
	if (DFS1(x + 1))
		return 1;
	s[p[x - 1] - 1] = 2;
	return DFS1(x + 1);
}
int main() {
#ifdef ONLINE_JUDGE
	std::ios::sync_with_stdio(false);
	std::cin.tie(nullptr);
	std::cout.tie(nullptr);
#endif
	std::string s;
	std::cin >> n >> d >> s;
	for (int i = 1; i <= n; ++i) {
		::s[i - 1] = s[i - 1] - 'a' + 1;
		if (s[i - 1] == 'x')
			p.push_back(i);
	}
	std::cin >> m;
	for (int x, y; m--; ) {
		char a, b;
		std::cin >> x >> a >> y >> b;
		q.emplace_back(x, a - 'A' + 1, y, b - 'A' + 1);
	}
	if (!DFS1(1))
		std::cout << -1 << '\n';
	return 0;
}
