Trick 类

  • 一次性更改大量边的信息是复杂的。可以在不影响答案的情况下添加边。

  • 用 DP 代替大容斥。

  • （上一条的拓展） 数学技巧：求方案数的一种方式是用处理完的合法方案数乘上当前一步可选的方案数。

    这个很重要！！！尤其是对于 062 你这个数学菜鸡来说，每次做数数题都重新开一次这个页面有意义吗？？？给我记好了！！！

    具体可以去看看 这一篇 里的 Gerald and Giant Chess 那道题。

  • 来自 匹配数：挖掘题目隐藏性质。如果 DP 有后效性而无前效性，考虑倒序 DP。

  • 来自 染色：当判定难以考虑时，可反过来考虑如何构造，或许问题会迎刃而解。

  • 来自 分糖果：用 map 作计数用时可以用排序 + 统计连续相同元素数量达到相同的效果，而且基本没什么常数……

  • 来自 单词：包括但不限于 状压的小技巧，比如状态 111 要分别求解的话会拆成 100 + 010 + 001，复杂度很大，但可以只拆一个 100 出来，再加上剩余的 011，直接减到 $\mathcal O(1)$。
